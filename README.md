# MikhmonGo — Releases

MikhmonGo is a Flutter app for managing MikroTik hotspot/PPP routers on
Android.

This repository hosts **APK releases only** — no source code is published
here. See the [Releases](../../releases) page for the latest APK download.

## Install

1. Download the latest `.apk` from [Releases](../../releases).
2. On your Android device, enable "Install from unknown sources" if prompted.
3. Open the downloaded APK to install.

## Requirements

- Android 8.0 (API 26) or newer.
